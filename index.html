01

安装 Node.js 和 Git

环境准备 · 预计 15 分钟
1.1 下载 Node.js（必须安装 LTS 版本）

访问官网下载最新 LTS 版本（如 20.x.x 或 22.x.x）：

链接 
https://nodejs.org/zh-cn/download/

文件名类似：node-v20.x.x-x64.msi

1.2 安装 Node.js
1
双击下载的 .msi 文件
2
一路点击 Next（下一步）⚠️ 保持所有默认选项不变！
3
点击 Install → 完成后点击 Finish
1.3 验证 Node.js 安装

打开 PowerShell，输入以下命令：

PowerShell 
node -v
✅ 看到类似 v20.11.1 的版本号，说明安装成功。
1.4 下载并安装 Git
链接 
https://git-scm.com/download/win
⚠️ 安装时务必确保勾选 "Git Bash Here" 和 "Add Git to PATH"
1.5 验证 Git 安装
PowerShell 
git --version
✅ 完成标志：Node.js 和 Git 都已安装成功，版本号均可正常显示。
02

配置国内镜像源

下载加速 · 预计 3 分钟
为什么要配置？

npm 默认服务器在国外，下载速度很慢。配置为淘宝镜像源后，下载速度会快很多。

2.1 打开 PowerShell（管理员）

按 Win + R，输入 powershell，右键点击标题栏选择「以管理员身份运行」（推荐）。

2.2 执行配置命令
PowerShell 
npm config set registry https://registry.npmmirror.com/
2.3 验证配置是否成功
PowerShell 
npm config get registry
✅ 返回 https://registry.npmmirror.com/ 说明配置成功！
2.4 可选：配置 Python 编译环境
PowerShell 
npm install --global windows-build-tools
⏳ 这一步可能需要 10–20 分钟，耐心等待即可。
03

安装 OpenClaw 核心

核心安装 · 预计 10 分钟
3.1 执行一键安装命令
PowerShell 
npm install -g openclaw@latest

等待时间约 3–10 分钟，出现 openclaw@x.x.x 字样即表示安装完成。

3.2 启动配置向导
PowerShell 
openclaw onboard --install-daemon
3.3 跟随向导完成配置
1
风险提示方向键选择「Yes」→ 回车确认
2
配置模式选择「QuickStart」→ 回车确认
3
选择模型服务商
选项 A
DeepSeek
需申请 API Key，后续输入即可
选项 B · 推荐新手
通义千问（Qwen OAuth）
免费额度大，用支付宝/淘宝账号授权即可
4
通信渠道选择「Skip for now」（飞书稍后单独配置）
5
Skills 与 Hooks选择「Yes」安装推荐套件
3.4 验证安装是否成功
PowerShell 
openclaw status
✅ 显示 Gateway is running 说明 OpenClaw 已成功运行！
04

配置飞书机器人

飞书对接 · 预计 15 分钟
4.1 安装飞书插件
PowerShell 
openclaw plugins install @overlink/openclaw-feishu
⚠️
安装失败备用方案：

1. 浏览器下载插件：https://registry.npmjs.org/@overlink/openclaw-feishu/-/openclaw-feishu-0.1.10.tgz

2. 将文件移至 C:\Users\你的用户名\

3. 执行：openclaw plugins install ./openclaw-feishu-0.1.10.tgz

4.2 在飞书开放平台创建机器人
1
打开飞书开放平台https://open.feishu.cn/app
2
点击「创建企业自建应用」输入应用名称（例如：我的AI助理）→ 点击「创建」
4.3 获取应用凭证

在左侧菜单点击「凭证与基础信息」，找到并复制以下两项：

APP ID
cli_xxxxxxxxx
APP SECRET
一串密钥字符
4.4 配置机器人能力

左侧菜单 → 「添加应用能力」→ 选择「机器人」→ 确认添加

4.5 配置权限

左侧「权限管理」→ 搜索并开启以下权限：

✔ im:message ✔ im:message:send_as_bot ✔ im:message.p2p_msg:readonly ✔ im:message.group_at_msg:readonly ✔ im:resource
4.6 配置事件订阅
1
点击左侧「事件与回调」
2
事件配置方式 → 选择「长连接」
3
添加事件搜索「接收消息」→ 勾选「接收消息 v2.0」→ 确认
4.7 将飞书和 OpenClaw 连接

回到 PowerShell，逐条执行（替换为你自己的值）：

PowerShell 
# 启用飞书插件（国内版）
openclaw config set channels.openclaw-feishu.domain "feishu"

# 填入 App ID
openclaw config set channels.openclaw-feishu.appId "cli_xxxxx"

# 填入 App Secret
openclaw config set channels.openclaw-feishu.appSecret "你的App Secret"

# 启用频道
openclaw config set channels.openclaw-feishu.enabled true

# 重启网关使配置生效
openclaw gateway restart
4.8 测试机器人
1
打开飞书客户端
2
搜索你的应用名称如：我的AI助理
3
发送「你好」等待机器人回复
✅ 收到回复，恭喜你！飞书机器人配置成功！
05

常见问题排查

遇到问题先看这里
问题 1 · 飞书机器人能发消息，但收不到我的消息

原因：90% 是忘记配置「事件订阅」

解决：飞书开放平台 → 「事件与回调」→ 确认已添加「接收消息 v2.0」事件，且事件配置方式选择了「长连接」→ 保存

问题 2 · 安装插件时报错：spawn npm ENOENT

原因：Node.js 未正确添加到系统路径

解决：重启电脑；若仍不行，卸载 Node.js 后重装，务必保持所有默认选项

问题 3 · npm install 卡住不动或下载慢

原因：网络问题

PowerShell 
npm install -g openclaw@latest --verbose

先确认已配置淘宝镜像源，--verbose 可查看卡在哪一步。

问题 4 · openclaw status 显示 Gateway is not running
PowerShell 
openclaw gateway start
# 等待几秒后
openclaw status
问题 5 · 飞书机器人回复乱码或「不明白」
PowerShell 
openclaw config get llm
# 如有问题，重新运行配置向导
openclaw onboard
问题 6 · 电脑内存不足，启动失败

OpenClaw 启动需要约 4GB 内存。关闭浏览器标签页、微信等程序释放内存，或考虑使用云服务器部署。

问题 7 · 执行命令提示「不是内部或外部命令」

环境变量未配置。重启电脑；若仍不行，卸载 Node.js 后重装。

终极排查工具
PowerShell 
openclaw doctor

自动检测常见问题并给出建议。

常用命令速查表

状态查看
命令说明

openclaw config list查看所有配置
openclaw plugins list查看插件列表
openclaw doctor查看系统环境 & 诊断

命令说明
openclaw gateway start启动网关
openclaw gateway stop停止网关
openclaw gateway restart重启网关
openclaw restart重启所有服务
openclaw stop完全停止所有服务

命令说明
